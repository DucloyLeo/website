Techniques de résolution

Toutes les déductions utilisables pour résoudre un puzzle Tango, classées par difficulté. Aucune technique ne requiert de deviner : chaque placement découle d'une règle logique.

Techniques de base · Facile
⚖ Équilibre forcé Facile
Chaque ligne et chaque colonne doit contenir exactement 3 ☀ et 3 🌙.
Dès qu'une ligne ou colonne atteint 3 d'un symbole, toutes les cases vides restantes sont forcées vers l'autre symbole.

Exemple : une ligne contient déjà ☀ ☀ · 🌙 · ☀ — 3 soleils posés. Les deux cases vides deviennent obligatoirement 🌙.

🚫 Paire bloque la voisine Facile
Deux symboles identiques côte à côte (XX) forcent la case immédiatement adjacente à être l'autre symbole — sinon on obtiendrait trois identiques de suite, ce qui est interdit.

Règle générale : après une paire, la case suivante est toujours l'opposé.

🥪 Sandwich (X · X) Facile
Si le même symbole apparaît de chaque côté d'une case vide (X · X), cette case doit être l'opposé — sinon on crée un triplet interdit en la remplissant avec X.

C'est le pattern le plus fréquent dans les grilles faciles.

🔗 Contrainte directe (= / ×) Facile
Une contrainte = entre deux cases signifie qu'elles doivent porter le même symbole.
Une contrainte × signifie qu'elles doivent porter des symboles opposés.

Dès qu'une des deux cases est connue, l'autre est immédiatement déductible.

Techniques intermédiaires · Moyen
↔ Extrémités identiques → milieu opposé Moyen
Sur 4 cases consécutives, si les deux extrémités sont identiques (ex: ☀ · · ☀), les deux cases du milieu doivent être l'autre symbole.

Pourquoi ? Si une case du milieu était ☀, on aurait ☀ ☀ · ☀ ou ☀ · ☀ ☀ — deux paires obligent la 3ᵉ à être 🌙, et l'équilibre confirme le reste.

🔗🔗 Chaîne = = (trio identique) Moyen
Deux contraintes = consécutives (A = B = C) signifient que les trois cases sont identiques. Si l'une est connue, les deux autres sont forcées.

Applicable aussi verticalement. Attention : trois symboles identiques consécutifs étant interdits, cette situation ne peut se produire qu'avec des cases non adjacentes physiquement — vérifiez qu'elles ne forment pas de triplet.

🔗🔗 Chaîne × × (A et C identiques) Moyen
Deux contraintes × consécutives : A × B × C.
Si A ≠ B et B ≠ C, alors nécessairement A = C (tous deux différents de B, donc identiques entre eux).

Si B est connu, A et C sont tous les deux forcés vers l'opposé de B.

📦 Paire = comme unité Moyen
Une paire liée par = contribue deux fois le même symbole à sa ligne. Si la ligne a déjà 2 ☀ et contient une paire =, cette paire doit obligatoirement être deux 🌙 (sinon on dépasserait 3 ☀).

De même, une paire × apporte exactement un ☀ et un 🌙.

Techniques avancées · Difficile
❌ Élimination par contradiction Difficile
Suppose mentalement qu'une case vide reçoit le symbole X. Si cela mène immédiatement à une contradiction (triplet, dépassement d'équilibre, violation de contrainte), alors X est interdit et la case reçoit l'autre symbole.

C'est une déduction d'un niveau au-dessus des patterns visuels : elle nécessite de raisonner sur les conséquences d'un placement.

🏁 Ligne quasi-complète Difficile
Une ligne avec 5 cases remplies et 1 vide est trivialement forcée par l'équilibre (3 ☀ et 3 🌙 requis). Mais même avec 4 remplies, combiner l'équilibre avec la règle anti-triplet peut forcer les deux cases restantes sans ambiguïté.

Cherchez les lignes où les cases vides restantes ne laissent qu'une seule combinaison légale.

✚ Déduction croisée ligne × colonne Difficile
Une case appartient à la fois à une ligne et à une colonne. Parfois ni la ligne seule ni la colonne seule ne force la case — mais en combinant les deux contraintes, une seule valeur est légale.

Stratégie : quand une case semble libre, vérifiez si placer l'un ou l'autre symbole crée un problème dans la colonne (triplet ou déséquilibre) même si la ligne ne pose pas de problème.

🔄 Propagation en chaîne Difficile
Plusieurs contraintes reliées peuvent former une chaîne de déductions : résoudre A force B via une contrainte, B force C via une autre, etc.

Ces chaînes sont invisibles si on regarde les contraintes une par une. Il faut suivre le chemin : "si A = ☀, alors B = ☀ (car A = B), donc C = 🌙 (car B × C), donc D = 🌙 (car C = D)".

Sur les grilles difficiles, certaines cases ne se résolvent que via des chaînes de 3 contraintes ou plus.
