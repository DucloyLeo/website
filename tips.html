Techniques de résolution

Toutes les déductions utilisables pour résoudre un puzzle Tango, classées par difficulté. Aucune technique ne requiert de deviner : chaque placement découle d'une règle logique.

Techniques de base · Facile
⚖ Équilibre forcé Facile
Chaque ligne et colonne doit contenir exactement 3 ☀ et 3 🌙.
Dès qu'une ligne atteint 3 d'un symbole, toutes les cases vides restantes sont forcées vers l'autre symbole.

Exemple : ☀ ☀ · 🌙 · ☀ — 3 soleils → les deux vides deviennent 🌙.

🔗 Contrainte directe (= / ×) Facile
= entre deux cases : elles portent le même symbole.
× entre deux cases : elles portent des symboles opposés.

Dès qu'une des deux est connue, l'autre est immédiatement déductible. La contrainte peut être horizontale ou verticale.

Techniques intermédiaires · Moyen
↔ Distance 4 : la 6ᵉ case est forcée Moyen
Si deux cases séparées de 4 positions sont identiques ([0] et [4], ou [1] et [5]), la case à l'autre bout de la ligne est forcée vers l'opposé.

Il n'existe que 3 lignes légales contenant ce pattern, et dans les trois l'extrémité distante est toujours l'autre symbole.

🔗🔗 Chaîne × × (A et C identiques) Moyen

Techniques avancées · Difficile
❌ Élimination par contradiction Difficile
Suppose mentalement qu'une case reçoit X. Si cela crée immédiatement une contradiction (triplet, déséquilibre, contrainte violée), X est interdit → la case reçoit l'autre symbole.

Exemple : placer ☀ en [2] créerait un triplet ☀☀☀ — donc [2] = 🌙.

🏁 Ligne quasi-complète Difficile
Une ligne avec 5 cases remplies est trivialement forcée par l'équilibre. Avec seulement 4 remplies, combiner équilibre + anti-triplet peut forcer les deux cases restantes sans ambiguïté.

Cherchez les lignes où les cases vides ne laissent qu'une seule combinaison légale.

✚ Déduction croisée ligne × colonne Difficile
Une case appartient à une ligne et à une colonne. Ni l'une ni l'autre seule ne force la case — mais en combinant les deux, une seule valeur est légale.

Vérifiez si l'un des deux symboles crée un problème dans la colonne même si la ligne est neutre.

🔄 Propagation en chaîne Difficile
Plusieurs contraintes reliées forment une chaîne : A force B, B force C, C force D…

Invisible case par case. Il faut suivre le chemin : A=☀ → B=☀ (A=B) → C=🌙 (B×C) → D=🌙 (C=D).

Sur les grilles difficiles, certaines cases ne se résolvent que via des chaînes de 3 contraintes ou plus.
